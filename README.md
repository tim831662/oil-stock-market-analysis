# WTI Crude Oil vs S&P 500 Market Analysis

This repository contains a Python analysis script that explores the relationship between West Texas Intermediate (WTI) Crude Oil monthly returns and S&P 500 monthly returns. The current implementation uses a trailing 10-year window and cleans outliers before performing exploratory analysis and both linear and logistic regression.

## What the Script Does

The script `stat_analysis.py` performs the following steps:

1. **Data Loading & Preparation**
   * Loads data from a local Excel file.
   * Converts the `obs` column values like `2003M01` into proper `Date` objects.
   * Sorts the data by date.
   * Uses the last 120 rows (10 years of monthly data).
   * Computes monthly percentage returns for both `Oil` and `SP500`.
   * Drops rows with missing return values.

2. **Z-Scores & Outlier Removal**
   * Computes Z-scores for both `WTI_Return` and `SP500_Return`.
   * Removes points where either Z-score is outside the 3-standard-deviation range.
   * Saves the cleaned dataset to `cleaned_oil_sp500_data.csv`.

3. **Exploratory Data Analysis (EDA)**
   * Creates a scatter plot of WTI returns vs S&P 500 returns with a fitted trend line.
   * Creates a Pearson correlation heatmap.
   * Creates a dual-axis time series plot of WTI and S&P 500 monthly returns.

4. **Statistical Modeling**
   * Fits an Ordinary Least Squares (OLS) linear regression model to predict `SP500_Return` from `WTI_Return`.
   * Fits a logistic regression model to predict whether the S&P 500 return is non-negative (`SP500_Up`) from `WTI_Return`.
   * Prints the regression summary, confusion matrix, and classification report.

5. **Additional Visualization**
   * Displays a graphic confusion matrix for the logistic regression results.

## How to Replicate

1. Install the dependencies.

```powershell
python -m pip install pandas numpy matplotlib seaborn scipy statsmodels scikit-learn
```

2. Verify the data file path inside `stat_analysis.py`.

The script currently reads from:

```python
file_path = r'C:\Users\tim83\.cache\kagglehub\datasets\saurabhshahane\oil-price\versions\1\EE_Codes\SP500_OilDate for Prof. Narayan.xls'
```

Update that path if your local dataset is stored elsewhere.

3. Run the analysis.

```powershell
python stat_analysis.py
```

4. Review the outputs.

* The cleaned data is saved to `cleaned_oil_sp500_data.csv`.
* Plot windows will appear for scatter, heatmap, time series, and logistic confusion matrix.
* Text output is printed to the console with regression summaries and classification metrics.

## Notes

* The script uses the last 10 years of data by taking the final 120 rows of the loaded dataset.
* The logistic model predicts market direction as "up/flat" versus "down" based solely on WTI monthly returns.
* Outliers are removed using a 3-sigma filter on both return series.
